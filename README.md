# Installation and Usage
1. Clone Repo
2. Npm install
3. Open browser and go to  localhost:6969
4. Drag and drop mp3's onto the dropzone
5. Modify speed through input box by using %'s 
  - ex: 
  - 100% = normal play speed
  - 200% = twice as fast
  - 50%  = half speed
